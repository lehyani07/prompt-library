# تعليمات نظام التصميم (Design System Instructions)

عند تصميم أي صفحة أو مكون جديد، يرجى الرجوع دائمًا إلى نظام التصميم الموجود في المجلد `app/design/`، وتحديدًا:
1. **ملف التكوين**: `app/design/design.json` (يحتوي على الألوان، الخطوط، والقياسات).
2. **الصفحة المرجعية**: `app/design/page.tsx` (تعرض المكونات بشكل مرئي).

يرجى الالتزام باستخدام نفس الألوان، والخطوط، وأنماط المكونات (Buttons, Cards, Tables, Inputs) لضمان توحيد هوية الموقع.

# AI Prompt Instruction

When designing any new page or component, please always refer to the design system located in the `app/design/` directory, specifically:
1. **Configuration File**: `app/design/design.json` (contains colors, typography, tokens).
2. **Reference Page**: `app/design/page.tsx` (visual component showcase).

Ensure you use the exact same colors, fonts, and component styles (Buttons, Cards, Tables, Inputs) to maintain design consistency.
